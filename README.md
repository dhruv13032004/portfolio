## 🚀 Featured Projects

### 🔒 Smart Home Security System - ESP32
**Repository:** [smart-home-security](https://github.com/dhruv13032004/smart-home-security)

A comprehensive smart home security platform with advanced cybersecurity features.

**Tech Stack:**
- **Hardware:** ESP32, PIR Sensor, Ultrasonic HC-SR04, RC522 RFID, DHT11, IR Sensor, 4-Channel Relay
- **Language:** C++ (Arduino)
- **Architecture:** WebServer, REST API, Session-based Authentication

**Security Features:**
- ✅ SHA-256 password hashing & session tokens
- ✅ Brute-force protection (5 attempts → 5min lockout)
- ✅ Rate limiting (60 req/min, DoS prevention)
- ✅ RFID access control with whitelist management
- ✅ Real-time intrusion detection (PIR + Proximity)
- ✅ Comprehensive security event logging
- ✅ HTTPOnly cookies with IP-bound sessions

**Features:**
- Multi-sensor data collection (Temperature, Humidity, Light, Motion, Distance)
- Web-based control dashboard with real-time updates
- Relay control for appliance management
- Security logs with severity levels
- Statistical analytics (uptime, requests, auth metrics)

**Status:** ✅ v1.0 Released
